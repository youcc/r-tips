## Create a new tag
```
git tag -a v1.3.1 -m "My new version"
```
## Push the new tag to master branch
```
git push origin v1.3.1
```
# Show tag comments
```
git show v1.3.1
```
# Delete a local Git tag
```
git tag -d v1.3.1
```
# Delete a remote Git tag
```
git push --delete origin v1.3.1
```
# Delete a local tag
```
git tag -d v1.3.1
```
